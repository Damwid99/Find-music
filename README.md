# Find-music

## Opis

Aplikacja typu Shazam.

## Instalacja

1. Sklonuj to repozytorium:

```bash
https://github.com/Damwid99/Find-music.git
`````
2. Zainstaluj wymagane biblioteki za pomocą pliku requirements.txt:
```bash
pip install -r requirements.txt
`````

## Użycie

Uruchom aplikację:
```bash
python main.py
`````
